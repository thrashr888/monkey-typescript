# gi

working through a golang interpreter book

## Build

    $ go build

## Run

    $ go run main.go

or

    $ nvm use
    $ node --experimental-modules main.mjs

# Test

    $ go test .

or

    $ node --experimental-modules test.mjs

or watch for file changes

    $ npm i -g nodemon
    $ nodemon --experimental-modules test.mjs
